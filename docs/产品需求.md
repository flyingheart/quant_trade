# 量化回测工具 - 产品需求文档

## 1. 项目概述
本项目是一款**桌面端专业量化策略回测工具**，基于 Tauri + Rust + React 构建，提供极低占用、高性能、高安全性的量化回测体验。支持用户通过 JavaScript 编写自定义策略、自动生成参数面板、K线可视化、信号标注、深度回测、参数寻优、收益分析等全流程功能。

## 2. 核心功能需求

### 2.1 数据源模块
- 支持本地文件导入：CSV/JSON/Parquet 格式
- 支持 API 拉取行情数据（A股、港股、美股）
- 本地 DuckDB 缓存，支持增量更新
- 自动识别字段：time, open, high, low, close, volume
- 数据清洗：去重、补全缺失值、异常值过滤
- 按标的+时间建立索引，支持快速检索

### 2.2 K线与技术指标展示模块
- 支持多周期K线展示：1min/5min/15min/1H/4H/日线
- 主图支持蜡烛图、MA/EMA/BOLL 指标叠加
- 副图支持 MACD、RSI、KDJ、成交量 独立窗口
- 支持十字光标、悬浮详情、图表缩放、拖拽
- 回测完成后自动在K线标注买卖信号点
- 支持海量历史数据分片加载、无卡顿渲染

### 2.3 自动参数面板（Auto-UI）模块
- 解析 JS 脚本注释自动生成参数控制面板
- 支持参数类型：
  - number（滑块）
  - bool（开关）
  - select（下拉）
  - range（区间）
  - date（日期选择器）
- 支持参数快照、批量重置、配置保存/导入
- 调整参数实时同步到回测引擎，实时刷新信号

### 2.4 JS 策略脚本编辑模块
- 基于 Monaco Editor（VS Code 内核）
- 语法高亮、代码提示、格式化、错误提示
- 内置策略模板：均线金叉、RSI 超买超卖、布林带突破
- 策略沙箱安全隔离执行，无系统风险
- 支持 console.log 输出日志，支持断点调试

### 2.5 策略管理模块
- 本地文件系统存储策略（.js 文件）
- 数据库管理策略元数据（名称、标签、创建时间、修改时间）
- 支持策略导入/导出、复制、重命名、删除
- 支持策略搜索、标签筛选
- 策略版本快照（保存历史版本，支持回滚）
- 回测历史记录：保存最近 N 次回测结果
- 多组回测对比：参数对比、收益曲线叠加

### 2.6 回测引擎模块
- 逐K线回测 + 向量化计算（Rust + Polars）
- 支持自定义手续费、滑点、初始资金
- 支持仓位管理、止盈止损、买卖信号判断
- 输出完整交易记录、收益曲线、回撤统计
- 支持并行参数寻优、热力图生成

### 2.7 回测结果展示模块
- 收益概览：总收益、年化收益、最大回撤、胜率、盈亏比
- 收益曲线（资产净值）
- 交易明细列表（时间、价格、方向、盈亏、收益率）
- 运行日志、信号触发记录
- 支持按年/季/月分段分析

### 2.8 安全沙箱模块
- V8 独立隔离环境
- 内存限制、CPU 限制、超时强制终止
- 无文件读写、无网络访问
- 防止死循环、恶意代码

## 3. 业务流程
1. 用户导入/加载历史K线数据（本地文件或 API 拉取）
2. 编写/选择 JS 策略 → 自动生成参数面板
3. 调整参数 → 运行回测
4. Rust 后端执行沙箱策略 → 计算指标 → 模拟交易
5. 前端渲染K线信号、收益曲线、交易明细、分析报告
6. 支持参数寻优、多组回测对比、策略版本管理

## 4. 非功能需求
### 4.1 性能指标
- 回测速度：10万根K线回测 < 3 秒（M1/Intel i5 及以上）
- 内存占用：< 200MB（正常回测场景）
- 启动时间：< 2 秒（冷启动）
- 图表渲染：10万根K线加载 < 1 秒，拖拽/缩放 60fps

### 4.2 安全指标
- JS 策略完全沙箱化，系统级隔离
- 内存限制：64MB/策略实例
- CPU 限制：单核使用率 < 80%
- 超时终止：5 秒强制终止
- 无文件读写、无网络访问权限

### 4.3 体验指标
- 原生桌面流畅度，无卡顿、无延迟
- 参数调整实时响应 < 100ms
- 回测进度实时推送（每秒更新）
- 错误提示友好，定位准确

### 4.4 跨平台兼容
- 支持系统：Windows 10+ / macOS 11+ / Ubuntu 20.04+
- 支持架构：x86_64 / ARM64 (Apple Silicon)
- 数据格式：CSV/JSON/Parquet
- 行情市场：A股、港股、美股

### 4.5 错误处理
- 策略语法错误：Monaco 编辑器实时提示，标注行号
- 数据异常：自动检测并提示，支持跳过/修复
- 回测中断：支持断点续跑，保存中间状态
- 用户友好错误提示，避免技术术语

# 量化回测工具 - 技术方案文档

## 1. 整体架构

采用 **Tauri 2.0 原生桌面架构**（前后端不分离，内存级通信）
- 前端：React 18 + WebKit
- 后端：Rust + Tokio + Polars
- 沙箱：deno_core
- 数据库：DuckDB (本地分析)
- 通信：Tauri IPC（零拷贝、二进制传输）

架构链路：
```
前端 UI → Tauri IPC → Rust 计算引擎 → deno_core 沙箱执行策略 → 回测结果 → 前端渲染
```

数据流架构：
```
数据源层 → 清洗层 → 索引层 → 存储层
  ↓          ↓         ↓         ↓
CSV/API   去重/补全  时间索引  DuckDB/Parquet
JSON      字段映射   分片加载  本地缓存
```

## 2. 技术栈

### 前端
- 框架：React 18
- 状态管理：Zustand
- 图表：lightweight-charts (TradingView 官方)
- 编辑器：@monaco-editor/react
- 样式：TailwindCSS + clsx
- 交互：多窗口、原生窗口吸附、可折叠面板

### 后端（Rust）
- 框架：Tauri 2.0

- 沙箱：deno_core
- 存储：DuckDB + Parquet
- 异步：Tokio 多线程并行回测
- 错误处理：thiserror + anyhow

### 核心中间件
- JS 沙箱（deno_core 隔离）
- 指标缓存池（LRU）
- 沙箱实例池（预启动）
- 行情数据索引引擎

```rust
// src/data/loader.rs
use polars::prelude::*;
use std::path::Path;
use anyhow::Result;

pub struct DataLoader;

impl DataLoader {
    /// 从本地文件加载 K 线数据
    pub fn load_from_file(path: &str, format: &FileFormat) -> Result<Vec<KlineBar>> {
        let df = match format {
            FileFormat::Csv => CsvReader::from_path(path)?.finish()?,
            FileFormat::Json => JsonReader::from_path(path)?.finish()?,
            FileFormat::Parquet => ParquetReader::new(std::fs::File::open(path)?).finish()?,
        };
        
        Self::df_to_klines(&df)
    }

    /// 从 API 拉取数据（A股/港股/美股）
    pub async fn load_from_api(config: &DataConfig) -> Result<Vec<KlineBar>> {
        match &config.source {
            DataSource::RestApi { url, .. } if url.contains("tencent") => {
                // 腾讯财经 API（港股/美股）
                Self::fetch_tencent_finance(config).await
            }
            DataSource::RestApi { url, .. } if url.contains("tdx") => {
                // 通达信接口（A股）
                Self::fetch_tdx_kline(config).await
            }
            _ => bail!("不支持的数据源"),
        }
    }

    /// 腾讯财经 API 数据拉取（港股/美股）
    /// 示例: https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param=sh000001,day,,,320,qfq
    async fn fetch_tencent_finance(config: &DataConfig) -> Result<Vec<KlineBar>> {
        let client = reqwest::Client::new();
        let interval_map = match config.interval {
            Interval::Min5 => "5",
            Interval::Min15 => "15",
            Interval::Min30 => "30",
            Interval::Day1 => "day",
            _ => bail!("腾讯财经不支持该周期"),
        };
        let url = format!(
            "https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param={},{},,,320,qfq",
            config.symbol, interval_map
        );
        let resp = client.get(&url).send().await?.json::<serde_json::Value>().await?;
        Self::parse_tencent_klines(&resp)
    }

    /// 通达信 K 线数据拉取（A股）
    /// 通过 mootdx 封装的 HTTP 接口获取
    async fn fetch_tdx_kline(config: &DataConfig) -> Result<Vec<KlineBar>> {
        let client = reqwest::Client::new();
        let interval_map = match config.interval {
            Interval::Min5 => "5",
            Interval::Min15 => "15",
            Interval::Min30 => "30",
            Interval::Day1 => "day",
            _ => bail!("通达信不支持该周期"),
        };
        let url = format!(
            "https://tdx.example.com/api/kline?symbol={}&interval={}&limit=1000",
            config.symbol, interval_map
        );
        let resp = client.get(&url).send().await?.json::<Vec<serde_json::Value>>().await?;
        Self::parse_tdx_klines(&resp)
    }

    /// 数据清洗：去重、补全缺失值、校验异常
    pub fn clean(data: &mut Vec<KlineBar>) {
        data.sort_by_key(|k| k.timestamp);
        data.dedup_by_key(|k| k.timestamp);
        data.retain(|k| k.open > 0.0 && k.high > 0.0 && k.low > 0.0 && k.close > 0.0);
        data.retain(|k| k.high >= k.low && k.high >= k.open && k.high >= k.close);
    }

    fn df_to_klines(df: &DataFrame) -> Result<Vec<KlineBar>> {
        let timestamp = df.column("time")?.datetime()?.into_iter()
            .map(|v| v.map(|ts| DateTime::from_timestamp(ts / 1000, 0).unwrap()))
            .collect::<Vec<_>>();
        let open = df.column("open")?.f64()?.into_no_null_iter().collect::<Vec<_>>();
        let high = df.column("high")?.f64()?.into_no_null_iter().collect::<Vec<_>>();
        let low = df.column("low")?.f64()?.into_no_null_iter().collect::<Vec<_>>();
        let close = df.column("close")?.f64()?.into_no_null_iter().collect::<Vec<_>>();
        let volume = df.column("volume")?.f64()?.into_no_null_iter().collect::<Vec<_>>();

        let klines = (0..timestamp.len()).map(|i| KlineBar {
            timestamp: timestamp[i].unwrap(),
            open: open[i],
            high: high[i],
            low: low[i],
            close: close[i],
            volume: volume[i],
            symbol: "UNKNOWN".into(),
            interval: Interval::Day1,
        }).collect();

        Ok(klines)
    }

    fn parse_binance_klines(resp: &[Vec<serde_json::Value>]) -> Result<Vec<KlineBar>> {
        resp.iter().map(|k| {
            Ok(KlineBar {
                timestamp: DateTime::from_timestamp_millis(k[0].as_i64().unwrap()).unwrap(),
                open: k[1].as_str().unwrap().parse()?,
                high: k[2].as_str().unwrap().parse()?,
                low: k[3].as_str().unwrap().parse()?,
                close: k[4].as_str().unwrap().parse()?,
                volume: k[5].as_str().unwrap().parse()?,
                symbol: "UNKNOWN".into(),
                interval: Interval::Day1,
            })
        }).collect()
    }

    fn interval_to_str(interval: &Interval) -> &'static str {
        match interval {
            Interval::Min1 => "1m", Interval::Min5 => "5m", Interval::Min15 => "15m",
            Interval::Hour1 => "1h", Interval::Hour4 => "4h", Interval::Day1 => "1d",
        }
    }
}
```

### 3.3 JS 沙箱模块

#### 技术选型

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| `deno_core` | 成熟、TypeScript 支持、内置权限控制 | 体积大、依赖多 | **推荐** |
| `v8` crate | 轻量、底层控制 | API 复杂、需手动管理 Isolate | 备选 |
| `quickjs` | 纯 Rust、无外部依赖 | 性能差、ES6 支持弱 | 不推荐 |

**最终方案**：`deno_core` + 权限沙箱

#### 沙箱架构

```
Rust 主进程
    │
    ├── 沙箱实例池 (预启动 3 个 Isolate)
    │       │
    │       ├── Isolate #1 → 策略执行
    │       ├── Isolate #2 → 空闲备用
    │       └── Isolate #3 → 空闲备用
    │
    ├── 权限控制器 (deno_core::Permissions)
    │       ├── 禁止文件读写
    │       ├── 禁止网络访问
    │       └── 限制内存 64MB
    │
    └── 看门狗 (超时 5 秒强制终止)
```

#### 核心实现

```rust
// src/sandbox/strategy_runner.rs
use deno_core::{JsRuntime, RuntimeOptions, Extension, OpState};
use anyhow::{Result, bail};
use std::time::{Duration, Instant};
use tokio::time::timeout;

#[derive(Debug, Clone, serde::Serialize)]
pub struct StrategyContext {
    pub klines: Vec<KlineBar>,
    pub params: serde_json::Value,
    pub current_index: usize,
}

#[derive(Debug, Clone, serde::Deserialize)]
pub enum Signal {
    Buy { price: f64, amount: f64 },
    Sell { price: f64, amount: f64 },
    Hold,
}

pub struct StrategyRunner {
    runtime: JsRuntime,
    timeout: Duration,
}

impl StrategyRunner {
    pub fn new() -> Result<Self> {
        let ext = Extension::builder("strategy_api")
            .ops(vec![
                ("op_log", op_log::decl()),
                ("op_get_kline", op_get_kline::decl()),
                ("op_get_param", op_get_param::decl()),
            ])
            .build();

        let mut runtime = JsRuntime::new(RuntimeOptions {
            extensions: vec![ext],
            ..Default::default()
        });

        runtime.execute_script("init.js", r#"
            globalThis.strategy = {
                log: (msg) => Deno.core.ops.op_log(msg),
                getKline: (index) => Deno.core.ops.op_get_kline(index),
                getParam: (key) => Deno.core.ops.op_get_param(key),
                MA: (data, period) => {/* 均线计算 */},
                CROSS: (a, b) => {/* 交叉判断 */},
            };
            globalThis.BUY = "BUY";
            globalThis.SELL = "SELL";
            globalThis.HOLD = "HOLD";
        "#)?;

        Ok(Self {
            runtime,
            timeout: Duration::from_secs(5),
        })
    }

    pub async fn execute(&mut self, script: &str, context: StrategyContext) -> Result<Signal> {
        self.set_context(&context)?;

        let result = timeout(self.timeout, async {
            self.runtime.execute_script("strategy.js", script)?;
            let result = self.runtime.execute_script("call_run.js", "run()")?;
            let signal: Signal = serde_v8::from_v8(self.runtime.v8_isolate(), result)?;
            Ok(signal)
        }).await;

        match result {
            Ok(Ok(signal)) => Ok(signal),
            Ok(Err(e)) => bail!("策略执行错误: {}", e),
            Err(_) => bail!("策略执行超时 ({}s)", self.timeout.as_secs()),
        }
    }

    fn set_context(&mut self, ctx: &StrategyContext) -> Result<()> {
        let ctx_json = serde_json::to_string(ctx)?;
        self.runtime.execute_script("set_context.js", &format!(
            "globalThis.__strategyContext = {};", ctx_json
        ))?;
        Ok(())
    }
}

#[op]
fn op_log(state: &mut OpState, msg: String) {
    println!("[策略日志] {}", msg);
    state.borrow_mut::<Vec<String>>().push(msg);
}

#[op]
fn op_get_kline(state: &mut OpState, index: usize) -> Result<serde_json::Value> {
    let ctx = state.borrow::<StrategyContext>();
    if index >= ctx.klines.len() {
        bail!("K线索引越界: {}", index);
    }
    Ok(serde_json::to_value(&ctx.klines[index])?)
}

#[op]
fn op_get_param(state: &mut OpState, key: String) -> Result<serde_json::Value> {
    let ctx = state.borrow::<StrategyContext>();
    Ok(ctx.params.get(&key).cloned().unwrap_or(serde_json::Value::Null))
}
```

#### 沙箱实例池

```rust
// src/sandbox/pool.rs
use std::sync::Arc;
use tokio::sync::Mutex;
use std::sync::atomic::{AtomicUsize, Ordering};

pub struct SandboxPool {
    runners: Vec<Arc<Mutex<StrategyRunner>>>,
    capacity: usize,
}

impl SandboxPool {
    pub fn new(capacity: usize) -> Result<Self> {
        let mut runners = Vec::with_capacity(capacity);
        for _ in 0..capacity {
            runners.push(Arc::new(Mutex::new(StrategyRunner::new()?)));
        }
        Ok(Self { runners, capacity })
    }

    pub async fn acquire(&self) -> Arc<Mutex<StrategyRunner>> {
        static INDEX: AtomicUsize = AtomicUsize::new(0);
        let idx = INDEX.fetch_add(1, Ordering::Relaxed) % self.capacity;
        self.runners[idx].clone()
    }
}
```

```typescript
// src/store/index.ts
import { create } from 'zustand';
import { invoke } from '@tauri-apps/api/core';
import type { AppState, BacktestResult, KlineBar } from './types';

interface Store extends AppState {
  setKlines: (klines: KlineBar[]) => void;
  setStrategyCode: (code: string) => void;
  updateParam: (key: string, value: any) => void;
  runBacktest: () => Promise<void>;
  resetParams: () => void;
  setActiveTab: (tab: AppState['activeTab']) => void;
  toggleLeftPanel: () => void;
  toggleRightPanel: () => void;
}

export const useStore = create<Store>((set, get) => ({
  klines: [],
  symbol: 'BTC-USDT',
  interval: '1d',
  strategyCode: '// 编写你的策略\nfunction run() {\n  return HOLD;\n}',
  strategyParams: { maFast: 5, maSlow: 10, rsi: 14 },
  isRunning: false,
  progress: 0,
  result: null,
  error: null,
  activeTab: 'overview',
  leftPanelCollapsed: false,
  rightPanelCollapsed: false,

  setKlines: (klines) => set({ klines }),
  setStrategyCode: (code) => set({ strategyCode: code }),
  updateParam: (key, value) => set((state) => ({
    strategyParams: { ...state.strategyParams, [key]: value }
  })),

  runBacktest: async () => {
    set({ isRunning: true, progress: 0, error: null });
    try {
      const { strategyCode, strategyParams, klines } = get();
      const result: BacktestResult = await invoke('run_backtest', {
        script: strategyCode,
        params: strategyParams,
        klines,
      });
      set({ result, isRunning: false, progress: 100 });
    } catch (err: any) {
      set({ error: err.message || '回测失败', isRunning: false });
    }
  },

  resetParams: () => set((state) => ({
    strategyParams: { maFast: 5, maSlow: 10, rsi: 14 }
  })),

  setActiveTab: (tab) => set({ activeTab: tab }),
  toggleLeftPanel: () => set((s) => ({ leftPanelCollapsed: !s.leftPanelCollapsed })),
  toggleRightPanel: () => set((s) => ({ rightPanelCollapsed: !s.rightPanelCollapsed })),
}));
```

```tsx
          {isRunning ? `回测中 ${progress}%` : '运行回测'}

        {isRunning && (
          <div className="mt-2 h-1 bg-[#1f2335] rounded overflow-hidden">
            <div
              className="h-full bg-[#7aa2f7] transition-all duration-300"
```

```rust
use serde::Serialize;

#[derive(Serialize)]
pub struct ApiResponse<T> {
    pub success: bool,
    pub data: Option<T>,
    pub error: Option<String>,
}

impl<T> ApiResponse<T> {
    pub fn ok(data: T) -> Self {
        Self { success: true, data: Some(data), error: None }
    }
    
    pub fn err(msg: String) -> Self {
        Self { success: false, data: None, error: Some(msg) }
    }
```
